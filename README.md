# Тестовое задание для компании "Новый Диск"

## Установка проекта

В корне приложения:
```
yarn install
```
Далее
```
cd server
npm i
```

### Далее открываем 2 терминала,
на одном в корне:
```
yarn serve
```
на другом:
```
cd server
npm start
```
